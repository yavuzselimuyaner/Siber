# XSS Notes

## Definition
Cross-Site Scripting (XSS) allows an attacker to inject malicious JavaScript into a web application so it executes in a victim's browser.

## Types
- Stored XSS
- Reflected XSS
- DOM-based XSS

## Notes
- Input sanitization and output encoding are essential.
- Payloads often depend on the surrounding HTML or JavaScript context.
- Risk can be reduced through CSP, proper encoding, and validation.
