# OverTheWire Bandit Notes

This writeup provides a high-level summary of the Bandit wargame rather than a full step-by-step walkthrough.

## Overview
Bandit is an excellent starting point for learning basic Linux commands, file permissions, processes, and simple shell usage.

## Notable Challenging Levels
### Levels 10-12
- Understanding file types is essential.
- Tools such as `file`, `strings`, and `base64` are commonly useful.

### Levels 13-14
- Working with SSH private keys becomes critical.
- Switching to the correct user context is often required.

### Levels 15-16
- Connecting with tools like `openssl` introduces basic service interaction.
- Interpreting the expected protocol correctly is important.

## Key Takeaways
- Basic Linux commands should be practiced regularly.
- File permissions and process handling are foundational concepts.
- Each level encourages a different problem-solving approach.
