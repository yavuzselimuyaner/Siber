# Cybersecurity Notes

This repository is a structured collection of cybersecurity study notes, writeups, and small security tools.

## About This Repository
- It is intended for continuous learning in web security, reconnaissance, and basic exploitation concepts.
- Notes are organized by platform and topic for easier navigation.
- Small Python tools developed during practice are kept alongside the notes.

## Navigation
- Start with [portswigger/README.md](portswigger/README.md) for PortSwigger lab notes.
- Use [tryhackme/README.md](tryhackme/README.md) for future TryHackMe progress.
- Review [overthewire-bandit/writeup.md](overthewire-bandit/writeup.md) for Bandit-related summaries.
- Explore the custom tools in [tools](tools).

## Goals
- Learn core cybersecurity concepts systematically
- Document vulnerability classes and techniques clearly
- Build small practical tools for security testing
- Track progress over time and revisit prior material
