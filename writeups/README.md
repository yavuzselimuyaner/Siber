# Writeups

Penetrasyon testi ve web güvenliği lab çözümleri.

## İçindekiler

- [`htb/`](htb/) — HackTheBox makine writeup'ları
- [`portswigger/`](portswigger/) — PortSwigger Web Security Academy lab'leri (OWASP Top 10)
- [`../picoCTF/`](../picoCTF/) — picoCTF challenge çözümleri

## Format

Her writeup `TEMPLATE.md` formatını takip eder:

1. **Recon** — keşif çıktıları (nmap, gobuster, ffuf vb.)
2. **Enumeration** — servis bazlı detaylı tarama
3. **Exploitation** — kullanılan zafiyet ve payload
4. **Privilege Escalation** — yetki yükseltme adımları
5. **Lessons Learned** — çıkarılan dersler

## Uyarı

Tüm pratikler kendi lab ortamımda veya yetkili platformlarda (HackTheBox, PortSwigger, picoCTF) gerçekleştirildi.
